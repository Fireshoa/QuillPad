# QuillPad
A script that opens a powerful note taking tool.
## To use:
1. Find the "source.js" file
2. Copy the contents
3. Press F12 or CTRL+SHIFT+I to open the inspect page
4. Make sure you are on the "Console" tab
5. Copy + Paste the source.js contents into the console
I have yet to find a better way to do this. Sorry!
